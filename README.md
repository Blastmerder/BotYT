# How to install
for install you need python, and uv, or pip for installing uv.

## First steps
### check baze

type in consle
```bash
python -version
```
if it wrote error, you need to install python,

then wrote
```bash
uv -version
```

if get error install uv by
```bash
pip install uv
```

## Second step

run the install.sh file, for installing dependensys, yiu can do this by
```bash
sh install.sh
```

## Final steps

run the bot by
```bash
sh run.sh
```
or
```bash
uv run bot.py
```
